## 超量限速不停机：具体是怎么运作的

这个机制从 2025 年 2 月开始覆盖洛杉矶 Pro 和 EB 全系套餐。逻辑是：每月流量配额用完之后，不关机，改为限速继续运行。

限速幅度按套餐分档：

- **WEE / MALIBU / CORONA 系列**：超量后限速 2Mbps
- **TINY / Pocket / PalmSpring / Starter 系列**：超量后限速 4Mbps
- **MINI / MICRO 系列**：超量后限速 8Mbps
- **MEDIUM / LARGE / GIANT 系列**：超量后限速 10Mbps

4Mbps 对于跑 SSH、维护后台、小流量网站基本够用，不会突然断线。如果你是做 API 服务、小型建站，这个设计确实省了很多麻烦。

T1 WEE 系列（包括香港和东京节点）超量后的限速更宽松：香港 / 东京 WEE 限速到 50Mbps，洛杉矶 WEE 限速到 100Mbps——实际上接近"超量无限"的体验，刷个视频、跑后台任务都不影响。

另外有一个 **Premium Unmetered 系列（LAX.Pro.u）**，直接就是不限流量 + CN2 GIA，对于视频站、下载站、CDN 源站这类流量消耗大的业务，是最彻底的解法。

不夸张。这套机制把"流量焦虑"这个问题，从根上解掉了大半。
